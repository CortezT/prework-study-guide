HTML

  • The "head" element contains information about the webpage.
  • The "body" element represents the visible content shown to the user.
  • The "id" element identifies elements and selects them.
  • The "h1-h6" elements indicate the text size.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A border property assigns a border with a width.
  • Any HTML element assigned to the class, card, will gain these property assignments from the CSS.
  • A box-shadow CSS property adds shadow effects around an element's frame. You can set multiple effects separated by commas.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: commit all files in the working branch.
  • git commit -m: associate a message with our commit.
  • git pull origin "main": pull in the base branch.
  • git push origin "feature/add-html": push up all of our changes.

JavaScript

  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • The code x++ is a shortcut way of writing x+1.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • To call a function, we use the name of the function in our JavaScript file where we want the code to execute.
